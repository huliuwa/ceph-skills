获取软件
有几种获取 Ceph 软件的方法。最简单和最常见的方法是获取软件包，即添加存储库供 Advanced Package Tool (APT) 或 Yellowdog Updater, Modified (YUM) 等软件包管理工具使用。您也可以从 Ceph 存储库检索预编译的软件包。最后，您可以检索 tarball 或克隆 Ceph 源代码存储库并自行构建 Ceph。

安装软件
一旦您获得了 Ceph 软件（或添加了存储库），安装软件就很容易了。要在集群中的每个Ceph 节点上安装软件包，请使用软件包管理工具。如果您打算安装 Ceph 对象网关或 QEMU，则应为 RHEL/CentOS 和其他使用 Yum 的发行版安装 Yum Priorities。

升级软件
随着 Ceph 新版本的发布，您可以升级集群以利用新功能。在升级集群之前，请阅读升级文档。有时升级 Ceph 需要您遵循升级序列。
